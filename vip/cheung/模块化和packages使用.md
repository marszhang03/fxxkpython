
# 模块名和packages

学习的方法、类、变量，以及所写的可执行代码，都可以纳入一个模块里面来。

1. import 模块名称


```python
import math

print(math.sqrt(9))
```

    3.0
    

2. from 模块名 import 具体函数


```python
from math import sqrt

print(sqrt(9))
```

    3.0
    

所谓packages，可以直接理解为一个目录，有时候选哟引用多个模块，但是模块之间可能存在于不同的目录中，当我们调用它的时候，为了更好的区分，我们就可以引用包下的模块名，这样就好区分了。

格式：

__from 包名.模块名 import 具体函数__
